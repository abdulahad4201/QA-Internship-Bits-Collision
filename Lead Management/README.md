# Lead Management Mobile App – Testing Project

This project showcases my testing work on a Lead Management mobile application, focusing on identifying bugs across multiple core modules.

---

## 📌 Project Overview

* **Project Name:** Lead Management Mobile App
* **Type:** Mobile Application Testing
* **Testing Type:** Manual Testing
* **Focus Area:** Functional & UI Testing

---

## 📱 Testing Scope

In this project, I performed testing on various modules, including:

* Sign In
* Sign Up
* Home Page
* Forgot Password
* Change Password
* Follow-Up Module
* Other core features

---

## 🐞 Bug Reports

Created detailed bug reports for issues found across different modules, covering:

* Functional issues
* Input validation errors
* UI/UX inconsistencies
* Navigation-related problems

Each report includes steps to reproduce, expected vs actual results, and severity levels.

---

## 🧠 Key Learnings

* Experience in mobile application testing
* Identifying bugs across multiple interconnected modules
* Improving attention to detail in UI and functionality
* Writing structured and clear bug reports

---

## 🛠 Tools Used

* Manual Testing
* Mobile Device Testing

---

## 🚀 Conclusion

This project improved my ability to test mobile applications and handle complex workflows involving multiple features, making my bug reporting more effective and professional.
