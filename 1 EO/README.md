
# EO Earth Optimization – Website Testing Project

This project highlights my work on testing the EO Earth Optimization website, focusing on authentication features.

---

## 📌 Project Overview

* **Project Name:** EO Earth Optimization
* **Type:** Web Application Testing
* **Testing Type:** Manual Testing
* **Focus Area:** Authentication (Login & Sign Up)

---

## 🔐 Testing Scope

During this project, I performed testing on:

* Login functionality
* Sign Up (Registration) process

---

## 🧪 Test Cases

Designed and executed test cases for the login module, covering:

* Valid and invalid login scenarios
* Input validation checks
* Error message verification

---

Created bug reports for issues identified in the **Sign Up process**, including:

* Input validation issues
* Functional errors
* UI-related problems

Each report includes clear steps, expected vs actual results, and severity levels.
