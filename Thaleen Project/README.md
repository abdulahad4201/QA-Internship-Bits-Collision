
# Thaleen Website Testing Project

This project represents my work during QA Internship, where I performed **manual testing** on the Thaleen website and created detailed bug reports.

---

## 📌 Project Overview

* **Project Name:** Thaleen Website
* **Type:** Web Application Testing
* **Testing Type:** Manual Testing
* **Focus Area:** Functional Testing & UI/UX Issues

The main objective of this project was to identify bugs and ensure the website functions correctly across different languages.

---

## 🌍 Multi-Language Testing

In this project, I tested the website in multiple languages to ensure proper functionality and accurate translations. The testing included:

* English Version
* Arabic Version
* Additional Language Support

Special attention was given to:

* Language switching functionality
* Text alignment and layout issues
* Translation accuracy
* UI responsiveness in different languages

---

## 🐞 Bug Reports

This repository contains **3 detailed bug reports** based on testing performed on different language versions of the website.

Each bug report includes:

* Clear Bug Title
* Supporting Screenshots

## 🧠 Key Learnings

* Improved skills in identifying and documenting bugs
* Gained experience in multi-language (localization) testing
* Learned how UI behaves differently in RTL (Right-to-Left) languages like Arabic
* Enhanced attention to detail in functional and UI testing
